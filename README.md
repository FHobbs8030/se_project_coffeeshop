
# ☕ Triple Peaks Coffee Shop

A modern **responsive coffee shop landing page** built as part of the **TripleTen Software Engineering Program**.  
This project demonstrates foundational front-end engineering skills including **semantic HTML, modular CSS architecture, and responsive design techniques**.

The project began as a static desktop layout and has since been **enhanced with responsive design**, allowing the interface to adapt smoothly across devices ranging from **320px mobile screens to 1440px desktop displays**.

# 📱 Responsive Design Enhancements

This project was upgraded to include **fully responsive behavior**, ensuring the layout works across modern devices.

Supported screen sizes include:

📱 **Mobile — 320px+**  
📱 **Large Mobile / Small Tablet — 480px+**  
💻 **Tablet / Laptop — 768px+**  
🖥 **Desktop — 1024px+**  
🖥 **Large Desktop — up to 1440px**

Responsive improvements include:

• Adaptive spacing and margins  
• Fluid layout containers  
• Media queries for device breakpoints  
• Scalable typography  
• Mobile-friendly navigation layout

# 🧩 Project Structure

The project uses a **BEM (Block-Element-Modifier) architecture** to keep styles modular and maintainable.

```
se_project_coffeeshop
│
├── blocks/                 # Individual BEM component styles
│   ├── about.css
│   ├── card.css
│   ├── footer.css
│   ├── form.css
│   ├── header.css
│   ├── menu.css
│   ├── nav.css
│   ├── recipes.css
│   └── reservation.css
│
├── pages/
│   └── index.css           # Main page layout styles
│
├── images/                 # Project images and SVG graphics
│
├── vendor/
│   └── normalize.css       # Cross-browser CSS normalization
│
├── index.html
├── favicon.ico
└── README.md
```
